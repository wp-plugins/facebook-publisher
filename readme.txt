Publish Wordpress posts on your Facebook pages.

== Description ==

Facebook Publisher is a simple plugin that lets you to publish your posts on Facebook pages directly from Wordpress administration.
If you want to increase your site social activity this is right plugin for you.

We also have a **PRO** version with additional features:

* Publish posts to the multiple Facebook pages in just one click
* Faster publishing & more optimized code
* Choose on which Facebook pages user (author, subscriber...) can publish post
* Faster and more efficient support

For any issues with the plugin, please open a support ticket and we will fix it ASAP!

== Installation ==

1. Unzip `facebook-publisher.zip` to your `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Create a Facebook APP at developers.facebook.com/apps/
4. Configure the plugin by going to the `Social Publisher Config` page
